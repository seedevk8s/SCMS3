## 1. 관리자 기능 개요

관리자는 시스템의 모든 기능에 접근할 수 있으며, 프로그램 관리, 신청 승인/거부, 마일리지 관리, 설문 관리 등의 권한을 가집니다.

  - 초기 비밀번호: `admin123`

## 3. 프로그램 관리

### 3.1 프로그램 목록 조회
1. 상단 메뉴에서 **관리자** → **프로그램 관리** 클릭
2. 전체 프로그램 목록 확인
3. 상태별 필터링: 예정/진행중/완료/취소

### 3.2 프로그램 등록
1. **프로그램 관리** 페이지에서 **새 프로그램** 버튼 클릭
2. 정보 입력:
   - **기본 정보**: 제목, 설명, 카테고리
   - **신청 기간**: 시작일, 종료일
   - **운영 기간**: 시작일, 종료일
   - **정원**: 최대 참여 인원
   - **마일리지**: 완료 시 지급할 마일리지 점수
   - **담당 부서**: 주관 부서명
3. 썸네일 이미지 URL 입력 (선택)
4. **등록** 클릭

### 3.3 프로그램 수정
1. 프로그램 목록에서 수정할 프로그램의 **수정** 버튼 클릭
2. 정보 수정
3. **저장** 클릭

### 3.4 프로그램 삭제
1. 프로그램 목록에서 삭제할 프로그램의 **삭제** 버튼 클릭
2. 확인 메시지 후 삭제
   - ⚠️ 신청자가 있는 프로그램은 삭제 불가 (상태를 "취소"로 변경)

---

## 4. 신청 관리

### 4.1 신청 내역 조회
1. **프로그램 관리** → 프로그램 선택 → **신청 내역** 탭 클릭
2. 신청자 목록 확인:
   - 학번, 이름, 학과, 학년
   - 신청일, 상태 (대기/승인/거부/취소)
3. 상태별 필터링

### 4.2 신청 승인
1. 대기 중인 신청의 **승인** 버튼 클릭
2. 확인 메시지 후 승인
3. 학생에게 승인 알림 자동 발송

### 4.3 신청 거부
1. 대기 중인 신청의 **거부** 버튼 클릭
2. 거부 사유 입력
3. **확인** 클릭
4. 학생에게 거부 알림 자동 발송 (사유 포함)

### 4.4 참여 완료 처리
1. 승인된 신청 중 프로그램이 종료된 건의 **완료** 버튼 클릭
2. 확인 메시지 후 완료 처리
3. **마일리지 자동 지급**

### 4.5 신청자 명단 다운로드
1. **신청 내역** 페이지에서 **Excel 다운로드** 버튼 클릭
2. 신청자 명단 Excel 파일 다운로드
   - 포함 정보: 학번, 이름, 학과, 학년, 연락처, 이메일, 신청일, 상태

---

## 5. 마일리지 관리

### 5.1 마일리지 규칙 관리
1. **관리자** → **마일리지 관리** 클릭
2. **규칙 관리** 탭 선택
3. 기존 규칙 수정 또는 새 규칙 추가
   - 활동 유형: PROGRAM, SURVEY, COUNSELING, PORTFOLIO 등
   - 활동명: 사용자에게 표시될 이름
   - 점수: 지급할 마일리지 점수
   - 활성 여부

### 5.2 마일리지 수동 지급/차감
1. **마일리지 관리** → **수동 지급** 탭 클릭
2. 정보 입력:
   - 학번 또는 이름으로 학생 검색
   - 점수 입력 (양수: 지급, 음수: 차감)
   - 활동명
   - 설명
3. **지급** 또는 **차감** 버튼 클릭

### 5.3 마일리지 랭킹 확인
1. **마일리지 관리** → **랭킹** 탭 클릭
2. 전체 학생 마일리지 순위 확인
3. Excel 다운로드 가능

---

## 6. 설문조사 관리

### 6.1 설문 생성
1. **관리자** → **설문 관리** → **새 설문** 클릭
2. 기본 정보 입력:
   - 제목, 설명
   - 시작일, 종료일
   - 필수 여부
3. **문항 추가**:
   - 문항 유형 선택: 객관식(단일/다중), 주관식, 평점(1-5)
   - 문항 내용 입력
   - 역량 연결 (역량 진단용 설문인 경우)
   - 필수 여부 설정
4. **저장** 클릭

### 6.2 설문 수정/삭제
1. **설문 관리** 페이지에서 설문 선택
2. **수정** 또는 **삭제** 버튼 클릭
   - ⚠️ 응답자가 있는 설문은 수정 시 주의

### 6.3 설문 활성화/비활성화
1. 설문 목록에서 **활성화 토글** 버튼 클릭
2. 비활성화된 설문은 학생에게 표시되지 않음

### 6.4 설문 응답 통계 확인
1. 설문 선택 → **통계** 탭 클릭
2. 응답 현황 확인:
   - 총 응답자 수
   - 문항별 응답 분포
   - 평균 점수 (평점 문항)
   - 그래프 시각화
3. **Excel 다운로드** 가능

---

## 7. 상담 관리 (상담사/관리자)

### 7.1 상담 신청 확인
1. **상담 관리** 메뉴 클릭
2. 대기 중인 상담 신청 목록 확인
   - 학생 정보, 상담 유형, 희망 일시, 주제

### 7.2 상담 승인/거부
1. 신청 건 선택
2. **승인** 또는 **거부** 선택
   - **승인**: 확정 일시, 장소, 비고 입력
   - **거부**: 거부 사유 입력
3. **확인** 클릭
4. 학생에게 알림 자동 발송

### 7.3 상담 기록 작성
1. 완료된 상담 건 선택
2. **기록 작성** 버튼 클릭
3. 정보 입력:
   - 상담 요약
   - 상세 내용
   - 권고사항
   - 후속 상담 필요 여부
4. **저장** 클릭
5. 마일리지 자동 지급 (학생)

### 7.4 상담 통계 확인
1. **상담 관리** → **통계** 탭 클릭
2. 통계 확인:
   - 월별 상담 건수
   - 상담 유형별 통계
   - 평균 만족도

---

## 8. 외부취업 가점 관리

### 8.1 승인 대기 목록 확인
1. **관리자** → **외부취업 가점** → **승인 대기** 탭 클릭
2. 학생이 등록한 외부취업 활동 확인
   - 학생 정보, 회사명, 근무 기간, 증빙 서류

### 8.2 활동 승인/거부
1. 활동 선택
2. 증빙 서류 확인
3. **승인** 또는 **거부** 선택
   - **승인**: 가점 자동 계산 및 부여
   - **거부**: 거부 사유 입력
4. **확인** 클릭

### 8.3 가점 규칙 관리
1. **외부취업 가점** → **규칙 관리** 탭 클릭
2. 취업 유형별 가점 설정:
   - 정규직: 기간별 가점
   - 계약직: 기간별 가점
   - 인턴: 기간별 가점
   - 파트타임: 기간별 가점
3. 규칙 수정 시 **저장** 클릭

---

## 9. 역량 진단 관리

### 9.1 역량 카테고리 관리
1. **관리자** → **역량 관리** → **카테고리** 탭 클릭
2. 역량 카테고리 추가/수정/삭제:
   - 의사소통, 문제해결, 창의성, 협업 등

### 9.2 역량 항목 관리
1. **역량 관리** → **역량 항목** 탭 클릭
2. 각 카테고리별 세부 역량 관리

### 9.3 학생별 역량 리포트 조회
1. **역량 관리** → **학생 리포트** 탭 클릭
2. 학생 검색
3. 역량 진단 결과 확인:
   - 각 역량별 점수
   - 평균 점수
   - 강점/약점 분석
   - 추천 프로그램

## 10. 알림 관리

### 10.1 전체 공지 발송
1. **관리자** → **알림 관리** 클릭
2. **새 공지** 버튼 클릭
3. 정보 입력:
   - 제목, 내용
   - 대상 설정: 전체/학과별/학년별
   - 링크 URL (선택)
4. **발송** 클릭

### 10.2 알림 발송 이력 확인
1. **알림 관리** → **발송 이력** 탭 클릭
2. 과거 발송한 알림 목록 확인
3. 발송 대상, 발송 시간 확인

### 12.1 데이터 백업

## 13. 자주 묻는 질문 (FAQ)

### Q1. 프로그램 신청을 실수로 거부했습니다. 다시 승인할 수 있나요?
A1. 아니요. 거부된 신청은 되돌릴 수 없습니다. 학생에게 재신청을 요청하세요.

### Q2. 마일리지를 잘못 지급했습니다. 어떻게 하나요?
A2. 마일리지 관리 → 수동 차감 기능을 사용하여 음수 점수로 차감하세요.

### Q3. 설문 응답 결과를 Excel로 다운로드할 수 있나요?
A3. 네, 설문 통계 페이지에서 "Excel 다운로드" 버튼을 클릭하세요.

### Q4. 외부취업 가점 규칙을 변경하면 기존 승인 건에도 적용되나요?
A4. 아니요. 규칙 변경은 변경 이후 승인되는 건부터 적용됩니다.
