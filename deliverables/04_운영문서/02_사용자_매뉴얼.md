# 사용자 매뉴얼

**프로젝트명**: 푸름대학교 학생성장지원센터 CHAMP (SCMS3)
**작성일**: 2025-11-18
**버전**: 1.0
**대상**: 학생

---

## 1. 시스템 소개

**SCMS3**는 푸름대학교 학생들의 성장을 지원하는 통합 시스템입니다. 비교과 프로그램 참여, 역량 진단, 마일리지 적립, 상담 신청, 포트폴리오 관리 등의 기능을 제공합니다.

---

## 2. 시작하기

### 2.1 접속 방법
- **URL**: http://scms.pureum.ac.kr
- **지원 브라우저**: Chrome, Firefox, Safari, Edge (최신 버전)

### 2.2 로그인

1. 메인 페이지 우측 상단의 **로그인** 버튼 클릭
2. **학번**과 **비밀번호** 입력
   - 최초 로그인 비밀번호: 생년월일 6자리 (예: 2003년 1월 1일 → `030101`)
3. **로그인** 버튼 클릭
4. 최초 로그인 시 비밀번호 변경 필수

### 2.3 비밀번호 찾기

1. 로그인 페이지에서 **비밀번호 찾기** 클릭
2. 학번, 이름, 생년월일 입력
3. 비밀번호가 생년월일(6자리)로 초기화됨

---

## 3. 주요 기능

### 3.1 프로그램 신청

#### 3.1.1 프로그램 목록 조회
1. 메인 페이지에서 프로그램 카드 확인
2. **전체보기** 클릭 시 모든 프로그램 조회
3. 필터 사용: 단과대, 학과, 카테고리로 필터링
4. 검색: 프로그램 제목으로 검색

#### 3.1.2 프로그램 상세 확인
1. 프로그램 카드 클릭
2. 상세 정보 확인:
   - 프로그램 설명
   - 신청 기간
   - 운영 기간
   - 정원 및 현재 신청 인원
   - 마일리지 점수
   - 담당 부서

#### 3.1.3 프로그램 신청
1. 프로그램 상세 페이지에서 **신청하기** 버튼 클릭
2. 신청 완료 메시지 확인
3. **나의 신청 내역**에서 상태 확인 (대기/승인/거부)

#### 3.1.4 신청 취소
1. **마이페이지** → **나의 신청 내역** 이동
2. 취소할 프로그램의 **취소** 버튼 클릭
3. 확인 메시지 후 취소 완료
   - ⚠️ 승인된 신청은 취소 불가

---

### 3.2 마일리지

#### 3.2.1 마일리지 조회
1. 상단 메뉴에서 **마일리지** 클릭
2. 내 총 마일리지 확인
3. 적립 내역 확인:
   - 활동 내용
   - 적립 점수
   - 적립 일자

#### 3.2.2 마일리지 획득 방법
- 비교과 프로그램 참여 완료: **100점**
- 역량 진단 설문 참여: **50점**
- 상담 완료: **50점**
- 포트폴리오 작성: **50점**
- 설문조사 참여: **30점**

#### 3.2.3 마일리지 랭킹
1. **마일리지** 페이지에서 **랭킹** 탭 클릭
2. 전체 학생 마일리지 순위 확인
3. 내 순위 및 점수 확인

---

### 3.3 역량 진단

#### 3.3.1 역량 진단 설문 참여
1. **설문조사** 메뉴 클릭
2. 진행 중인 역량 진단 설문 선택
3. 각 문항에 대해 1-5점 척도로 응답
4. 모든 필수 문항 응답 후 **제출** 클릭

#### 3.3.2 역량 진단 결과 확인
1. **마이페이지** → **역량 진단 결과** 이동
2. 각 역량별 점수 확인:
   - 의사소통
   - 문제해결
   - 창의성
   - 협업
3. 강점/약점 분석 확인
4. 추천 프로그램 확인

#### 3.3.3 추천 프로그램 확인
1. **프로그램** 페이지에서 **맞춤 추천** 탭 클릭
2. 내 역량 진단 결과 기반 추천 프로그램 확인
3. 추천 이유 확인

---

### 3.4 상담 신청

#### 3.4.1 상담사 목록 확인
1. **상담 신청** 메뉴 클릭
2. 상담사 목록 및 전문 분야 확인
3. 가용 일정 확인

#### 3.4.2 상담 신청하기
1. **상담 신청** 버튼 클릭
2. 정보 입력:
   - 상담 유형 선택 (진로/학업/심리/기타)
   - 희망 일시 선택
   - 상담 주제 입력
   - 상담 내용 입력
3. **신청** 버튼 클릭

#### 3.4.3 상담 내역 확인
1. **마이페이지** → **상담 내역** 이동
2. 상담 상태 확인 (대기/확정/완료/취소)
3. 확정된 상담의 일시 및 장소 확인

#### 3.4.4 상담 피드백 작성
1. 완료된 상담의 **피드백 작성** 버튼 클릭
2. 만족도 평가 (1-5점)
3. 피드백 내용 입력
4. **제출** 클릭

---

### 3.5 포트폴리오

#### 3.5.1 포트폴리오 생성
1. **포트폴리오** 메뉴 클릭
2. **새 포트폴리오 작성** 버튼 클릭
3. 정보 입력:
   - 제목
   - 설명
   - 카테고리
   - 태그
4. **저장** 클릭

#### 3.5.2 포트폴리오 항목 추가
1. 포트폴리오 상세 페이지에서 **항목 추가** 클릭
2. 항목 유형 선택:
   - 프로그램 참여
   - 자격증
   - 프로젝트
   - 대외활동
3. 상세 정보 입력
4. 증빙 파일 첨부 (선택사항)
5. **추가** 클릭

#### 3.5.3 포트폴리오 공유
1. 포트폴리오 상세 페이지에서 **공유** 버튼 클릭
2. 공개 범위 선택:
   - **PUBLIC**: 누구나 조회 가능
   - **UNLISTED**: 링크를 아는 사람만 조회
   - **PRIVATE**: 나만 조회
3. 공유 링크 복사

---

### 3.6 설문조사

#### 3.6.1 설문 참여
1. **설문조사** 메뉴 클릭
2. 참여 가능한 설문 목록 확인
3. 설문 선택 및 문항 응답
4. **제출** 클릭
5. 마일리지 자동 적립

---

### 3.7 외부취업 가점

#### 3.7.1 외부취업 활동 등록
1. **마이페이지** → **외부취업 가점** 메뉴 클릭
2. **활동 등록** 버튼 클릭
3. 정보 입력:
   - 회사명
   - 취업 유형 (정규직/계약직/인턴/파트타임)
   - 근무 기간
   - 직책
   - 업무 내용
4. 증빙 서류 첨부
5. **등록** 클릭

#### 3.7.2 가점 확인
1. 등록한 활동의 승인 상태 확인 (대기/승인/거부)
2. 승인된 활동의 가점 점수 확인
3. 총 획득 가점 확인

---

### 3.8 알림

#### 3.8.1 알림 확인
1. 상단 메뉴의 **종** 아이콘 클릭
2. 읽지 않은 알림 개수 확인
3. 알림 클릭 시 해당 페이지로 이동

#### 3.8.2 알림 유형
- 프로그램 신청 승인/거부
- 상담 일정 확정
- 마일리지 적립
- 설문조사 참여 요청
- 시스템 공지사항

## 4. 자주 묻는 질문 (FAQ)

### Q1. 비밀번호를 잊어버렸습니다.
A1. 로그인 페이지에서 "비밀번호 찾기"를 클릭하여 학번, 이름, 생년월일을 입력하면 비밀번호가 생년월일(6자리)로 초기화됩니다.

### Q2. 프로그램 신청 후 취소할 수 있나요?
A2. 승인 전에는 취소 가능하지만, 승인 후에는 취소할 수 없습니다. 담당 부서에 문의하세요.

### Q3. 마일리지는 어떻게 사용하나요?
A3. 마일리지는 각종 혜택 및 장학금 선발 시 활용됩니다. 자세한 내용은 학생성장지원센터에 문의하세요.

### Q4. 역량 진단은 몇 번 할 수 있나요?
A4. 학기당 1회 진행되며, 주기적으로 참여하여 역량 변화를 추적할 수 있습니다.

### Q5. 포트폴리오를 다른 사람과 공유하고 싶습니다.
A5. 포트폴리오 상세 페이지에서 "공유" 버튼을 클릭하여 공개 범위를 설정하고 링크를 복사하세요.
